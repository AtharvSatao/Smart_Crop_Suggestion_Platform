🌾 1. Farmer-to-Consumer Connection Platform (FCCP)
The Farmer-to-Consumer Connection Platform is a full-stack web application designed to bridge the gap between farmers and consumers by enabling direct online selling of agricultural products. It eliminates the need for intermediaries, ensures fair pricing, and provides consumers with access to high-quality, fresh produce directly from farmers.

📌 Key Features of FCCP
Role-based authentication (Farmer & Consumer)
Farmer Dashboard for adding, updating, and managing product listings
Consumer Dashboard for browsing and purchasing products
Shopping cart and order system
Product management with images, prices, and descriptions
Contact and feedback form
Secure database handling using MySQL

🛠️ Technologies Used (FCCP)
HTML, CSS, JavaScript
PHP
MySQL


🌱 2. Smart Crop Suggestion Platform (SCSP)
The Smart Crop Suggestion Platform is an advanced extension of FCCP that integrates a machine learning–based crop recommendation system. SCSP helps farmers choose the most suitable crop by analyzing real-time soil and environmental conditions.

📌 Key Features of SCSP
ML-powered crop prediction (94–96% accuracy)
Analyzes N, P, K, Temperature, Humidity, pH level, Rainfall
Provides best crop + top 2–3 alternative crops
Real-time PHP → Python communication using shell_exec()
User-friendly crop suggestion input page
Optional SMS alert system using Twilio API

🔬 Machine Learning Details
Algorithm: Random Forest Classifier
Language: Python
Libraries: Scikit-learn, NumPy, Pandas
Model Output:
Best suited crop
Probability-based alternative crops

🛠️ Technologies Used (SCSP)
Python
Scikit-learn, Pandas, NumPy
PHP
MySQL
HTML, CSS, JavaScript

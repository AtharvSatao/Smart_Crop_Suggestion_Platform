# 🌾 1. Farmer-to-Consumer Connection Platform (FCCP)

The **Farmer-to-Consumer Connection Platform** is a full-stack web application designed to connect farmers directly with consumers by enabling online selling of agricultural products. It removes intermediaries, ensures fair pricing, and gives consumers access to fresh, high-quality produce.

## 📌 Key Features of FCCP

- Role-based authentication (Farmer & Consumer)
- Farmer Dashboard for adding, updating, and managing product listings
- Consumer Dashboard for browsing and purchasing products
- Shopping cart and order system
- Product management with images, prices, and descriptions
- Contact and feedback form
- Secure database handling using MySQL

## 🛠️ Technologies Used (FCCP)

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL

# 🌱 2. Smart Crop Suggestion Platform (SCSP)

The **Smart Crop Suggestion Platform** is an advanced extension of FCCP that integrates a machine-learning–based crop recommendation system. It helps farmers select the most suitable crop based on real-time soil and environmental parameters.

## 📌 Key Features of SCSP

- ML-powered crop prediction (**94–96% accuracy**)
- Analyzes **N, P, K, Temperature, Humidity, pH level, Rainfall**
- Shows **best crop + top 2–3 alternative crops**
- Real-time PHP → Python communication using `shell_exec()`
- Simple, user-friendly input form for crop suggestions
- Optional SMS alerts using Twilio API

## 🔬 Machine Learning Details

- **Algorithm:** Random Forest Classifier
- **Language:** Python
- **Libraries:** Scikit-learn, NumPy, Pandas
- **Model Output:**
  - Best suited crop
  - Probability-based alternative crops

## 🛠️ Technologies Used (SCSP)

- **Machine Learning:** Python, Scikit-learn, Pandas, NumPy
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
